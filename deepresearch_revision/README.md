# Gemini Deep Research 요청 사항: 데이터 기반 재분석 및 논문 재작성

첨부된 **검증된 데이터(`all_subjects_complete_data.csv`, `subject_profiles.md`)를 바탕으로**, 기존 논문의 **섹션 4 (통계 분석)**와 **섹션 5 (개인차 분석)** 내용을 **새롭게 분석하여 한국어로 재작성(Rewrite)** 해주세요.

단순 번역이 아니라, 제공된 데이터를 근거로 **직접 분석한 결과**를 서술해야 합니다.

## 📂 분석 대상 데이터

1.  **`all_subjects_complete_data.csv`**: 전체 8명의 검증된 정량 데이터 (심박수, RMSSD, EMG 등)
2.  **`subject_profiles.md`**: 개별 피험자 상세 프로파일 및 특이사항
3.  **`original_paper_section.tex`**: 재작성할 타겟 파일 (섹션 구조 유지용)

## 🎯 분석 및 작성 지침 (Instructions)

### 1. 데이터 기반의 새로운 분석 (Analysis First)

제공된 CSV 및 프로파일 데이터를 분석하여 다음 인사이트를 도출하고 논문에 반영해주세요:

- **전체 경향성**: Phase 2(탐색)와 Phase 3(추격)에서 전체적으로 어떤 생체 신호 변화가 있었는지 데이터(평균, 편차)를 근거로 서술하세요.
- **특이 패턴 발견**: 데이터에서 발견되는 특이한 개인차를 찾아내어 심층 분석하세요.
    - **Subject E2 (Takamiya)**: **[중요]** Phase 3의 심박수/RMSSD (29 bpm, 4766 ms)는 **아티팩트(Movement Artifact)에 의한 측정 오류**임이 확인되었습니다. 이 수치를 생체 반응으로 해석하지 마세요. 대신, **102 μV의 높은 근긴장도(EMG)**를 근거로 피험자가 극도의 공포 상태에서 격렬한 신체 반응을 보였음을 서술하세요. (기존 서맥 분석 폐기)
    - **Subject F (Sensei)**: 주관적 공포는 낮으나(1점) 심박수와 근긴장도가 높은 현상을 **"무의식적 긴장"** 또는 **"해리적 반응"**으로 해석하세요.
    - **Subject D (Ishikawa)**: 안정적인 데이터 패턴을 **"저반응(Low Responder)"** 유형으로 분류하여 비교 분석하세요.

### 2. 논문 재작성 (Rewrite in Korean)

- **언어**: **한국어**
- **구조**: `original_paper_section.tex`의 섹션 구조(`\section`, `\table` 등)는 그대로 유지하세요.
- **내용**: 위에서 도출한 분석 내용을 바탕으로, 기존 텍스트에 얽매이지 말고 **데이터가 말하는 사실**을 중심으로 명확하고 논리적으로 서술해주세요. 통계 수치는 반드시 첨부된 CSV의 정확한 값을 인용해야 합니다.

## ⚠️ 주의사항
